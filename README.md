# Preview Goods Size

サイズ表記テキストを実際の物理サイズで可視化するChrome拡張機能です。

## 概要

このChrome拡張機能は、WebページやPDFに記載されているサイズ表記テキスト（例：`H100mm×W148mm`）を選択して右クリックするだけで、そのサイズを画面上に実寸大でプレビュー表示できます。

商品の寸法を確認する際に、実際のサイズ感を画面上で体験できるため、購入前の検討やデザイン検証に役立ちます。

## 機能

- **サイズ表記自動認識**：複数のフォーマットに対応したサイズ表記パターンマッチング
  - `H100mm×W148mm×D50mm` 形式
  - `W148mm×H100mm×D50mm` 形式
  - 順序が異なる形式にも対応

- **実寸プレビュー**：認識したサイズを画面上に実寸大で表示

- **デバイスピクセル比対応**：複数のディスプレイ設定に対応
  - 1920x1080 23.8インチ (FHD)
  - 2560x1441 27.9インチ (150% スケーリング)
  - 3840x2160 27.9インチ (4K)

## 使い方

1. Chrome拡張機能をインストール
2. Webページでサイズテキストをハイライトして選択
3. 右クリックメニューから「**実寸サイズをプレビュー**」を選択
4. 選択したサイズが画面上に実寸大で表示されます

## インストール方法

1. `src` ディレクトリの内容を確認
2. Chromeの設定から「拡張機能を管理」を開く
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトの `src` フォルダを選択

## ファイル構成

```txt
src/
├── manifest.json           # 拡張機能の設定ファイル
├── content_script.js       # ページコンテンツ内で実行されるスクリプト
├── service_worker.js       # バックグラウンドで実行されるスクリプト
└── icon16.png             # 拡張機能のアイコン
```

### manifest.json

Manifest v3形式の設定ファイル。拡張機能の基本情報、権限、スクリプトの指定を定義します。

### content_script.js

- サイズテキストのパターンマッチングと解析
- デバイスピクセル比の計算
- ミリメートル単位を物理単位に変換
- プレビュー表示のレンダリング

### service_worker.js

- コンテキストメニューの作成と管理
- ユーザーのクリックイベント処理
- コンテンツスクリプトとのメッセージング

## 対応フォーマット

以下のサイズ表記フォーマットに対応しています：

- `H100mm×W148mm×D50mm` (高さ、幅、奥行き)
- `W148mm×H100mm×D50mm` (幅、高さ、奥行き)
- `100mm×148mm×50mm` (順序不問)
- `H100mm×W148mm` (高さ、幅のみ)
- `W148mm×H100mm` (幅、高さのみ)
- `100mm×148mm` (2次元サイズ)
